# KAZARBUILD — Технологическое руководство

PWA-приложение для строительной компании KAZARBUILD.

## Как запустить

1. Загрузите проект на GitHub:
   - Создайте репозиторий с именем `kazarbuild`
   - Загрузите все файлы
   - В Settings → Pages выберите ветку `main`

2. Сайт будет доступен по адресу:
   `https://ВАШ_ЛОГИН.github.io/kazarbuild`

## Как добавлять карты

Редактируйте файл `data/cards.json`. Структура карты:

```json
{
  "id": "card_уникальный_id",
  "category_id": "cat_1",
  "slug": "nazvanie-karty",
  "title": { "ru": "Название карты" },
  "description": { "ru": "Краткое описание" },
  "content": {
    "ru": {
      "blocks": [
        { "type": "h2", "text": "Заголовок раздела" },
        { "type": "p",  "text": "Текст параграфа" },
        { "type": "ul", "items": ["Пункт 1", "Пункт 2"] },
        { "type": "checklist", "items": ["Шаг 1", "Шаг 2"] }
      ],
      "navigation": [
        { "text": "Связанная карта", "card_id": "card_002" }
      ]
    }
  }
}
```

## Как добавлять объекты (фото)

Редактируйте `data/projects.json`:

```json
{
  "id": "proj_001",
  "title": "Название объекта",
  "area": 100,
  "location": "Город",
  "duration": "2 месяца",
  "year": 2025,
  "tags": ["Под ключ"],
  "description": "Описание",
  "cover": "photos/proj001/cover.jpg",
  "photos": ["photos/proj001/1.jpg", "photos/proj001/2.jpg"]
}
```

## Контакты

Отредактируйте `data/contacts.json` — замените номера телефонов, адрес и соцсети.

## Структура файлов

```
kazarbuild/
├── index.html          ← основное приложение
├── sw.js               ← сервис-воркер (офлайн)
├── manifest.json       ← PWA манифест
├── data/
│   ├── categories.json ← 32 категории
│   ├── cards.json      ← тех.карты
│   ├── projects.json   ← объекты/галерея
│   └── contacts.json   ← контакты компании
├── icons/
│   ├── icon-192.png
│   └── icon-512.png
└── README.md
```
